# Agent Skills

A collection of AI Agent skills designed to enhance productivity and automate testing workflows.

## Current Skills

### 1. Facebook Login Test Planner
- **Location**: `.agent/skills/facebook-test-planner/SKILL.md`
- **Purpose**: Assists in creating short and concise test plans for the Facebook login page using a specific template.
- **Workflow**: Guides the user through gathering environment details and generating a comprehensive test plan following a standard 8-section template.

### 2. Tweet Master
- **Location**: `.agent/skills/crafting-tweets/SKILL.md`
- **Purpose**: Research and craft high-engagement, clickbaity, and CTA-friendly tweets for X.com.

## How to Use these Skills

These skills are designed to be used by AI agents (like Antigravity). To use a skill:
1. Ensure the `.agent/skills` folder is in your workspace.
2. Ask the agent to use a specific skill (e.g., "Use the Facebook Login Test Planner skill").
3. Follow the interactive prompts provided by the agent.
